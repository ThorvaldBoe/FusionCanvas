## Source PRD

# FC-0803 - Workflow Templates

## Summary

Workflow Templates provide reusable structures for common production styles, stores, niches, and campaigns.

## Requirements

- Users can create templates for recurring workflows.
- Templates can define suggested groups, statuses, tags, metadata needs, or queues.
- Templates can be applied to new work.
- Templates can be edited, duplicated, archived, or deleted.
- Applying a template should be reviewable before it changes the workspace.

## Acceptance Criteria

- A user can save a recurring workflow structure.
- A user can apply a template to new work.
- A user can adjust templates over time.

## Out of Scope

- Fully automated production
- Marketplace-specific automation
- Team template sharing
