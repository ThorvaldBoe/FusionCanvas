## Source PRD

# FC-0304 - Batch Asset Import

## Summary

Batch Asset Import lets users bring many files into FusionCanvas and connect them to the right work.

## User Need

Creators often produce many source files, exports, or mockups at once and need efficient asset organization.

## Requirements

- Users can import multiple files in one operation.
- Assets can be assigned a purpose or type.
- Users can map files to listings manually or with simple matching support.
- Failed or unmatched files should be visible.
- Imported assets should preserve source context and relationships.

## Acceptance Criteria

- A user can import many assets together.
- A user can map imported files to listings.
- A user can identify files that still need assignment.
- Imported assets appear in relevant listing contexts.

## Out of Scope

- Advanced image analysis
- Automatic perfect matching
- Cloud sync
- Asset deduplication
