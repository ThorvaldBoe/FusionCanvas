## Source PRD

# FC-0403 - Niche AI Context

## Summary

Niche AI Context uses stored niche knowledge to make AI assistance more relevant.

## Requirements

- Niche context can include audience, tone, humor, style, constraints, and successful patterns.
- AI workflows can use selected niche context when generating or refining work.
- Users can review and edit niche context.
- Users can avoid sending irrelevant or sensitive context.
- Niche context should remain human-authored and editable.

## Acceptance Criteria

- AI output can be grounded in a selected niche.
- A user can see which context informs a prompt.
- Updating niche context changes future AI assistance.

## Out of Scope

- Automatic trend research
- Autonomous niche creation
- Performance-based learning
