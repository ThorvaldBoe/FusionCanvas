## Source PRD

# FC-0409 - Critique and Scoring

## Summary

Critique and Scoring reviews a concept, phrase, and graphic direction for strengths, weaknesses, and improvement options.

In the Basic Concept Tool, scoring can appear as the quality indicator in the center of the design triangle.

## Requirements

- Users can request critique for a listing or concept.
- Critique can evaluate idea, phrase, graphic alignment, audience fit, risks, and clarity.
- Critique can identify the weakest design triangle element and suggest which node to improve next.
- Scoring can produce an overall readiness indicator for the current concept.
- Scoring should account for designs that intentionally omit phrase or graphic.
- Scores should support judgment, not replace it.
- Users can save critique notes to the listing history.
- Users can ignore or revise based on critique.
- Scoring should not automatically advance, reject, or archive a concept.

## Acceptance Criteria

- A user can request critique for a listing.
- Critique identifies actionable improvement areas.
- Critique can be shown in the Basic Concept Tool as a triangle quality indicator.
- Saved critique remains connected to the listing.

## Out of Scope

- Automatic rejection
- Sales prediction
- Legal validation
