## Source PRD

# FC-0703 - Listing Performance View

## Summary

Listing Performance View shows performance history and derived metrics for an individual listing.

## Requirements

- Users can view performance data for one listing.
- The view can show raw metrics and derived metrics.
- Users can see performance by marketplace where available.
- Trends over time should be understandable.
- Performance should be tied back to creative context.

## Acceptance Criteria

- A user can inspect performance for a listing.
- A user can see sales, traffic, and engagement signals where available.
- The view helps decide whether to revise, pause, expand, or archive.

## Out of Scope

- Niche-level analytics
- Automated recommendations
- Ad campaign management
