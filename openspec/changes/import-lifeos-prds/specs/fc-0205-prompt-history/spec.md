## Source PRD

# FC-0205 - Prompt History

## Summary

Prompt History preserves AI prompts, outputs, and related context.

## User Need

Creators need to remember which prompts produced useful ideas, phrases, images, critiques, or listing text.

## Requirements

- Users can save prompt inputs and outputs.
- Prompts can be associated with stores, niches, listings, concepts, designs, or assets.
- Prompt type should be identifiable.
- Provider and model information can be recorded when available.
- Prompt history should preserve useful rejected or superseded outputs.
- Prompt records should support later reuse and review.

## Acceptance Criteria

- A user can view prompts related to a listing.
- A user can preserve useful prompt outputs.
- Prompt context remains connected to the creative work it affected.
- Prompt history can support future AI workflows.

## Out of Scope

- Live AI execution
- Prompt template library
- Provider configuration
- Automatic critique or scoring
