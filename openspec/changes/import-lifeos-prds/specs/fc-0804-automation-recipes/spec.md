## Source PRD

# FC-0804 - Automation Recipes

## Summary

Automation Recipes define user-approved sequences for repetitive workflows.

## Requirements

- Users can define or choose a sequence of supported actions.
- Recipes can operate on selected listings, groups, assets, or queues.
- Users can preview what a recipe will do.
- Recipes should require approval before making meaningful changes.
- Recipe results should be recorded.

## Acceptance Criteria

- A user can run a recipe for a repetitive workflow.
- A user can review intended actions before execution.
- Completed recipe actions are traceable.

## Out of Scope

- Unrestricted scripting
- Background scheduling
- Autonomous publishing
