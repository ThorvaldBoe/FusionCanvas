# FC-0704 - Niche Performance View

## Summary

Niche Performance View aggregates performance by niche, group, tag, phrase type, style, or campaign.

## Requirements

- Users can view performance summaries for a niche.
- Aggregation can include listings, groups, tags, or campaigns.
- The view should highlight patterns across related work.
- Users can drill into listings behind the summary.
- Performance should support learning, not just reporting.

## Acceptance Criteria

- A user can see which listings contribute to niche performance.
- A user can compare groups or tags within a niche.
- The view helps identify promising or weak areas.

## Out of Scope

- Automated market research
- Forecasting
- Ad platform optimization
