# FC-0406 - Phrase Generation

## Summary

Phrase Generation suggests verbal hooks, variations, and templates for product concepts.

In the Basic Concept Tool, phrase generation is triggered by selecting the phrase node of the design triangle.

## Requirements

- Users can generate phrases from a listing, idea, niche, or concept.
- Phrase generation should use the active item, store, niche, topic path, idea, graphic direction, tags, metadata, and nearby work where available.
- Phrase generation can be used to fill an empty phrase node or improve a weak phrase.
- Phrase generation is optional because some designs are intentionally graphic-only.
- Outputs can include alternatives and variation notes.
- Users can save selected phrases to listings or phrase libraries where available.
- Users can reject or archive poor phrases.
- Rejected phrase suggestions can be used as negative guidance for later generation where practical.
- Generated phrases should support human review for originality and fit.

## Acceptance Criteria

- A user can generate phrase options for a concept.
- A user can save a selected phrase to a listing.
- A user can regenerate phrase options without changing the existing accepted phrase.
- Unused phrases do not overwrite current listing content.

## Out of Scope

- Trademark clearance
- Guaranteed originality
- Automated marketplace keyword optimization
