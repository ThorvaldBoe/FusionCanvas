# FC-0706 - Experiment Tracking

## Summary

Experiment Tracking helps creators compare deliberate variations across groups, campaigns, prompts, styles, or marketplaces.

## Requirements

- Users can define an experiment.
- Experiments can include related listings, groups, tags, prompts, or marketplace products.
- Users can record hypothesis, variables, dates, and notes.
- Performance can be reviewed against experiment context.
- Completed experiments should remain available for learning.

## Acceptance Criteria

- A user can define an experiment around a creative or publishing question.
- A user can attach relevant listings or groups.
- A user can review outcome notes and performance signals.

## Out of Scope

- Statistical testing engine
- Automatic experiment design
- Ad platform split testing
