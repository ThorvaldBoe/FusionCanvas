# FC-0508 - Sample Plugin

## Summary

Sample Plugin demonstrates the expected plugin structure and extension points.

## Requirements

- The sample includes a valid manifest.
- The sample demonstrates discovery.
- The sample demonstrates service or command contribution.
- The sample demonstrates settings where useful.
- The sample demonstrates plugin data or event usage where practical.
- The sample should be understandable to future plugin authors.

## Acceptance Criteria

- A contributor can inspect the sample plugin to understand the plugin model.
- The sample plugin can be discovered by FusionCanvas.
- The sample demonstrates at least one useful extension point.

## Out of Scope

- Production marketplace integration
- Complex UI
- Full plugin SDK documentation
