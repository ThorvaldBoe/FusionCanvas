## Source PRD

# FC-0705 - Phrase and Style Pattern Tracking

## Summary

Phrase and Style Pattern Tracking identifies recurring creative patterns in successful and unsuccessful designs.

## Requirements

- Listings can be analyzed or grouped by phrase traits, style traits, tags, or notes.
- Users can identify which patterns correlate with performance.
- Patterns should be reviewable by niche, group, or store.
- Users can record observations about successful and unsuccessful patterns.
- Pattern tracking should support future idea generation and critique.

## Acceptance Criteria

- A user can compare performance across phrase or style patterns.
- A user can record what seems to work or fail.
- Pattern insights remain connected to the relevant niche or store.

## Out of Scope

- Fully automatic creative judgment
- Guaranteed causal analysis
- Trend scraping
