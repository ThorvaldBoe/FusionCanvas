## Source PRD

# FC-0602 - Export Package Builder

## Summary

Export Package Builder creates local packages containing artwork, mockups, and listing metadata.

## Requirements

- Users can select listings for export.
- Packages can include selected assets and metadata.
- Users can see missing required information before export.
- Package structure should be predictable.
- Exporting should not require marketplace integration.

## Acceptance Criteria

- A user can build a local export package for selected listings.
- The package includes relevant assets and metadata.
- Missing files or metadata are surfaced before completion.

## Out of Scope

- Direct API publishing
- Automated product creation
- Fulfillment setup
